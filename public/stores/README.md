# Логотипы магазинов для ценников

Положите сюда логотипы, имена файлов должны совпадать с `logo` в `lib/stores.ts`:

- `arzonchi.png`
- `chinnibaza.png`
- `engarzon.png`
- `abusaxiy.png`

Лучше PNG/SVG с прозрачным фоном, высота ~120px и больше.
Пока файла нет — на ценнике показывается название магазина текстом.
